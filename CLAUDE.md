# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Tooling

- **Linter/Formatter**: [ruff](https://docs.astral.sh/ruff/) is used for Python linting and formatting.
  - Lint: `ruff check .`
  - Format: `ruff format .`
  - Fix auto-fixable issues: `ruff check --fix .`
